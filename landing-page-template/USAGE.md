# Quick Usage Guide

## To Create a New Landing Page from This Template

### Step 1: Copy the Template
```bash
# From your workspace root
cp -r landing-page-template my-new-project
cd my-new-project
```

### Step 2: Install Dependencies (if needed)
```bash
npm install
```

### Step 3: Start Development
```bash
npm run dev
```

### Step 4: Open in Browser
Visit: http://localhost:3000

## What You Get

✅ Next.js 14 with TypeScript  
✅ Tailwind CSS configured  
✅ 8 shadcn/ui components ready to use  
✅ Lucide React icons  
✅ Framer Motion for animations  
✅ Example page showing everything in action  

## Next Steps

1. Edit `app/page.tsx` to build your landing page
2. Add more components: `npx shadcn@latest add [component-name]`
3. Customize colors in `app/globals.css`
4. Add your assets to the `public/` folder

That's it! You're ready to build.
